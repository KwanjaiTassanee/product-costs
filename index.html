🌶️ ระบบบริหารการผลิตและคำนวณต้นทุนน้ำพริกกากหมู

มาตรฐานรอบผลิตฐาน: มันหมู 20 กก. บรรจุได้ 100 ถุง (น้ำหนักรวม 5.5 กก.)

✨ เลือกรสชาติน้ำพริกที่ต้องการผลิต

🛒 1. มีวัตถุดิบเท่านี้ ผลิตได้กี่ถุง?

💡 ผลลัพธ์การประเมิน:
จากมันหมูติดหนัง 0 กก.
จะสามารถผลิตน้ำพริกได้ประมาณ: 0 ถุง
(น้ำหนักเนื้อน้ำพริกรวมสำเร็จประมาณ 0 กก.)

🎯 2. ตั้งเป้าผลิต X ถุง ต้องเตรียมวัตถุดิบเท่าไหร่?

📝 รายการวัตถุดิบที่ต้องจัดเตรียม (รส):
    ต้นทุนวัตถุดิบรวมทั้งหมด: 0 บาท
    (เฉลี่ยต้นทุนเฉพาะวัตถุดิบ 0 บาท/ถุง)

    📜 ประวัติการคำนวณและการผลิต
